FiltersApp
===============
___
## Short Information

This python app is used to add filters and image processing

![Alt text](img/readme/star-page.jpg)

## Setup

1. Install requirements

```
pip install -r requirements.txt
```

2. Run script

```
python main.py
```

##Filters and Processes 

###1. Crop Image
